# Waste Audit & Monitoring

## Overview
A web-based R Shiny application for collecting, managing, and visualizing solid waste data across multiple locations. This tool supports efficient waste audits by enabling structured data entry, real-time analytics, and centralized reporting.

## Features

### Structured Data Collection
- Spreadsheet-style input for segregation compliance and waste weights
- Multi-step submission workflow for guided data entry

### Interactive Dashboard
- Average daily waste generation
- Waste composition and segregation distribution
- Time-series and stacked visualizations using Plotly

### Admin Panel
- View, edit, and filter submitted data
- Download datasets in CSV format
- Automated summary table generation

### Data Management
- Local CSV-based storage with automatic initialization
- Robust handling of dates and numeric inputs
